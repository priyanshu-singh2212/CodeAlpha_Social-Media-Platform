# Social Media Platform

A mini social media web application that allows users to create profiles, share posts, interact through comments, and engage with other users using likes and follow features.

## 🚀 Features

* User Registration & Login Authentication
* User Profiles
* Create Posts
* View Posts in User Feed
* Comment on Posts
* Like Posts
* Follow / Unfollow Users
* Responsive User Interface
* Secure Backend API
* MongoDB Atlas Cloud Database Integration

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/priyanshu-singh2212/CodeAlpha_-Social-Media-Platform.git
```

### Navigate to Project

```bash
cd CodeAlpha_-Social-Media-Platform
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the backend directory and add:

```env
MONGODB_URI=your_mongodb_atlas_connection_string
PORT=5000
```

### Run the Server

```bash
npm start
```

or

```bash
node server.js
```
